YaMDb

База отзывов о фильмах, книгах и музыке

Зависимости

Для запуска приложения должны быть установлены Python3 и Docker.

Установка

1) поместить .env в дирректорию с проектом. 2) выполнить docker-compose up 3) в списке запущеных контейнеров (docker container ls) найти id контейнера api_yamdb_web 4) применить миграции: docker exec -it <api_yamdb_web container id> python manage.py migrate 5) создать пользователя: docker exec -it <api_yamdb_web container id> python manage.py createsuperuser 6) для заполнения БД начальными значениями применить фикстуры: docker exec -it <api_yamdb_web container id> python manage.py loaddata fixtures.json
