# yamdb

База отзывов о фильмах, книгах и музыке.

## установка

Склонировать репозиторий, разместить .env в директории с проектом.

```python
SECRET_KEY=******
EMAIL_HOST_USER=******
DB_ENGINE=******
DB_NAME=******
POSTGRES_USER=******
POSTGRES_******
DB_HOST=******
DB_PORT=******
```

Запустить docker-compose

```bash
docker-compose up
```

## использование

Документация к проекту
```
localhost/redoc
```
